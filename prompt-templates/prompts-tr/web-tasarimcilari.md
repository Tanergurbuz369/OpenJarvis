# Web Tasarımcıları Promptları (Web Designers)

20 Türkçe ChatGPT promptu. `[köşeli parantez]` içindeki yerler doldurulacak alanlardır.

1. [Belirli işletme türü] ana sayfası için web sitesi kopyası oluşturun.
2. [İşletme türü] için bir site haritası taslağı oluşturun.
3. [Belirli konular/hizmetler] hakkında sayfalar için meta açıklamaları yazın.
4. [Belirli hedef kitle] hedefleyen bir web sitesi için kullanıcı personaları geliştirin.
5. WCAG kurallarına dayalı bir web sitesi erişilebilirlik kontrol listesi oluşturun.
6. Web sitesi tasarımı keşif oturumu sırasında müşterilere sorulacak soruların bir listesini oluşturun.
7. Yaygın web sitesi hata mesajları için mikrometin yazın.
8. [Belirli tür] web sitesi için bir içerik hiyerarşisi oluşturun.
9. [Konu/işletme] hakkında bir web sitesi için etkileşimli öğeler fikirleri üretin.
10. Bir web sitesi genelinde marka tutarlılığını korumak için bir stil kılavuzu yazın.
11. [Belirli işletme türü] için bir web sitesi için bir renk paleti ve tipografi önerisi yazın.
12. Bir web sitesi analiz raporu için bir şablon oluşturun.
13. [Belirli hedef kitle] yapılan kullanılabilirlik testi için bir test senaryosu yazın.
14. [Belirli tür] web sitesi için kapsamlı bir içerik stratejisi geliştirin.
15. Müşterinin web sitesi tasarım geri bildirimini yapılandırmak için bir çerçeve oluşturun.
16. [Belirli sektör/işletme türü] için öne çıkan başarılı web sitesi tasarım örneklerinin bir analizini yazın.
17. [Belirli sayfa türü] kullanıcı deneyimi iyileştirme önerileri geliştirin.
18. Bir web sitesi için mobil kullanıcı arayüzü tasarım ilkeleri listesi oluşturun.
19. Yavaş yüklenen web sitelerini optimize etmek için stratejiler yazın.
20. [Belirli işletme hedefi] başarmak için bir açılış sayfası tasarım kılavuzu geliştirin.
